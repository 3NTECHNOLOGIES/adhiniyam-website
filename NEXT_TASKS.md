## Upload Workflow Reminder

For every new tool:
1. Upload new tool folder inside `/tools/`.
2. Confirm `index.html` and `og-image.png` exist.
3. Upload supporting files:
   - `assets/tools.json`
   - `assets/common.js`
   - `tools/index.html`
   - `sitemap.xml`
   - `README.txt`
4. Wait for Vercel deployment to show Ready.
5. Test live URL.
6. Submit sitemap.
7. Request indexing.
